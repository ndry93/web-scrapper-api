# web-scrapper-api

## Scraper Endpoint
`domain.com/api/scraper`

## API Docs
`domain.com/api/docs`

## Installation
```
1. npm i
2. npm run start:prd
```

## With Docker
Build Image
`docker build -t ndry93/web-scrapper-api .`

Run Image with local port :9999
`docker run 9999:3000 -d ndry93/web-scrapper-api`
